About us
Basic Chess is an html project for educational purposes. It shows the basic movements of each piece as well as the hidden moves within it.
The images seen within the website are taken from the internet. The project consist of 9-10 pages and the website is created by Carl Enrico Lim
The creator of the website likes to play chess and had the idea to create it for his project.
