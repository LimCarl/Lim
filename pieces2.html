   The Rook
   
   The rook is a powerful piece that can move horizontally and vertically using as much squares
   as it can. It resembles a castle like figure.	
	
   The Bishop
   
   Like the rook it is a powerful piece but instead of moving horizontally and vertically
   it can move diagonally using as much squares like the rook.
